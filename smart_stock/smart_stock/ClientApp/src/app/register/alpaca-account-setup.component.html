Create An Alpaca Account

We use a third party API service, Alpaca, for live stock market trading and data access. To get started with our application, you must create an account through Alpaca (it's free!), and submit you API key information into the forms below. We securely store these keys, and use them to make trades for you with our smart trading software! This is a one-time step, so you won't need to do this again!

Once you've created your account, be sure to select "Paper Trading" from the dropdown menu in the top left navbar. From there, click on the "view" button, on the right hand side of the dashboard, under the "Your API Keys" section. Copy the API Key ID, and the Secret Key into the forms below. These are for your eyes only! Once done, you're all set!
